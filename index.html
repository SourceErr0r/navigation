The Navigation router

Scene-Based Navigation for React and React Native

Scene-Based Navigation

Native apps have use scene-based navigation forever. The Navigation router is the first to bring it to the web. You give the Navigation router a list of your scenes. After you navigate to a scene, the Navigation router gets out of your way so you can build your UI however you want.
